📜 Included Scripts / Dahil Olan Scriptler

install_script.sh

EN: Installs to $HOME/Scripts, makes scripts executable, and adds to your PATH via .bashrc or .zshrc.

TR: Scriptleri $HOME/Scripts klasörüne kurar, çalıştırılabilir yapar ve PATH ayarını .bashrc veya .zshrc dosyasına ekler.

install_script.sh script1.sh script2.sh ...

✅ Skips installing if the script already exists in your system PATH
⚠️ Prompts before overwriting existing scripts in ~/Scripts
🛑 Skips copying a file onto itself
🔁 PATH is only updated if installation happens
🐚 Supports both Bash and Zsh
💡 Use bash install_script.sh if unsure

brave_singletonlock_delete.sh

Removes Brave’s SingletonLock file to fix launch issues.

brave_singletonlock_delete.sh

📥 Installation

git clone https://github.com/erenko147/useful-shell-scripts.git
cd useful-shell-scripts
install_script.sh brave_singletonlock_delete.sh

Or download and run manually:

wget https://raw.githubusercontent.com/erenko147/useful-shell-scripts/main/install_script.sh
bash install_script.sh

🧠 PATH Setup

The script appends the following block only if necessary:

if [[ ":$PATH:" != *":$HOME/Scripts:"* ]]; then
    export PATH="$HOME/Scripts:$PATH"
fi

So you can run scripts globally from any terminal.
