🧰 Useful Shell Scripts / Kullanışlı Bash Scriptler

EN: This repository contains simple yet helpful shell scripts which I created to automate small tasks while learning Unix scripting.
TR: Bu depo, günlük hayatta işe çok da yaramayacak fakat kendimi Unix script üzerinde geliştirmek ve karşılaştığım sorunlardan kurtulmak için geliştirdiğim scriptleri içerir.

📜 Included Scripts / Dahil Olan Scriptler

install_script.sh

EN: Installs one or more scripts to $HOME/Scripts, makes them executable with chmod +x, and adds the folder to your PATH automatically.

TR: Bir veya birden fazla scripti $HOME/Scripts dizinine kurar, chmod +x ile çalıştırılabilir hale getirir ve otomatik olarak PATH'e ekler.

Usage / Kullanım:

install_script.sh script1.sh script2.sh ...

💡 You can pass as many scripts as you like.
💡 İstediğiniz kadar script dosyasını aynı anda kurabilirsiniz.

✅ Tip: Add bash before the command to ensure it works even if not marked as executable:
bash install_script.sh script1.sh
🟢 Aynı zamanda başına bash ekleyerek garanti çalıştırabilirsiniz.

brave_singletonlock_delete.sh

EN: Removes the Brave browser's SingletonLock file that sometimes prevents it from launching.

TR: Brave tarayıcısının açılmasını engelleyen SingletonLock dosyasını siler.

brave_singletonlock_delete.sh

📥 How to Download / Nasıl İndirilir

Clone the repo / Depoyu klonlayın:

git clone https://github.com/erenko147/useful-shell-scripts.git
cd useful-shell-scripts

Run the installer / Kurulum scriptini çalıştırın:

install_script.sh brave_singletonlock_delete.sh

Download a single script / Tek script indirmek için:

wget https://raw.githubusercontent.com/erenko147/useful-shell-scripts/main/install_script.sh
bash install_script.sh

🧠 PATH Setup / PATH Ayarı (Otomatik)

The installer script adds this line to your .bashrc if not already present:

export PATH="$HOME/Scripts:$PATH"

This allows you to run your scripts globally from any terminal.
Böylece scriptleri her yerden terminal ile çalıştırabilirsiniz.
