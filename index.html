📜 Included Scripts / Dahil Olan Scriptler

install_script.sh

EN: Installs one or more scripts to $HOME/Scripts, makes them executable with chmod +x, and adds the folder to your PATH via .bashrc or .zshrc, depending on your shell.

TR: Bir veya birden fazla betiği $HOME/Scripts dizinine kurar, chmod +x ile çalıştırılabilir hale getirir ve kullandığınız kabuğa bağlı olarak .bashrc veya .zshrc dosyanıza bu dizini PATH olarak ekler.

install_script.sh script1.sh script2.sh ...

❗ Skips installing if the script already exists in your system PATH
🔄 Prompts before overwriting existing scripts in ~/Scripts
🛑 Skips copying a file onto itself
🔁 PATH is only updated if installation happens
🐚 Supports both Bash and Zsh
💡 Use bash install_script.sh if unsure

brave_singletonlock_delete.sh

EN: Removes the Brave browser lock file (SingletonLock) that, when closed improperly, prevents Brave from opening.

TR: Hatalı bir şekilde uygulamadan çıkıldığında Brave tarayıcısının açılmasını engelleyen SingletonLock adlı kilit dosyasını siler.

brave_singletonlock_delete.sh

travel_charge.sh

EN: This script temporarily sets your laptop battery to charge to 100% instead of the usual 80% limit, giving you maximum battery capacity for travel or extended unplugged use.

TR: Bu betik, seyahat veya uzun süre prizden bağımsız kullanım için maksimum pil kapasitesi sağlamak amacıyla laptop pilinizi geçici olarak normal %80 sınırı yerine %100'e kadar şarj edilecek şekilde ayarlar.

travel_charge.sh

⚠️ Note: This script only works on laptops that support battery charge thresholds through sysfs (like Huawei laptops, some ThinkPads, etc.) - it won't work on all laptop brands.
⚠️ Not: Bu betik sadece sysfs üzerinden pil şarj eşiklerini destekleyen laptoplarda çalışır (Huawei ve bazı ThinkPad'ler gibi) - tüm laptop markalarında çalışmaz.

📥 Installation / Kurulum

Clone the repository / Depoyu klonlayın:

git clone https://github.com/erenko147/useful-shell-scripts.git
cd useful-shell-scripts
install_script.sh brave_singletonlock_delete.sh travel_charge.sh

Or download and run manually / Ya da manuel indirip çalıştırın:

With wget:

wget https://raw.githubusercontent.com/erenko147/useful-shell-scripts/main/install_script.sh
bash install_script.sh

With curl:

curl -o install_script.sh https://raw.githubusercontent.com/erenko147/useful-shell-scripts/main/install_script.sh
bash install_script.sh

Or run directly without saving (curl only):

curl -s https://raw.githubusercontent.com/erenko147/useful-shell-scripts/main/install_script.sh | bash

EN: If you don't download install_script.sh you will have to manually manage your scripts' whereabouts and their permissions.
TR: Eğer install_script.sh dosyasını indirmezseniz, betiklerinizin yerini ve izinlerini manuel olarak yönetmeniz gerekecektir.

🧠 PATH Setup / PATH Ayarı

EN: The script adds the following block to your .bashrc or .zshrc file only if it doesn't already exist:

TR: Betik, aşağıdaki satırları .bashrc veya .zshrc dosyanıza sadece daha önce eklenmemişse otomatik olarak ekler:

# Add custom scripts path
if [[ ":$PATH:" != *":$HOME/Scripts:"* ]]; then
    export PATH="$HOME/Scripts:$PATH"
fi

EN: This allows you to run your scripts globally from any terminal.
TR: Bu sayede betiklerinizi terminalin herhangi bir yerinden çalıştırabilirsiniz.

EN: If no supported shell config file is found, the script will warn you and print the block to be added manually.
TR: Eğer desteklenen bir shell yapılandırma dosyası bulunamazsa, betik sizi uyarır ve manuel eklemeniz gereken satırları gösterir.
